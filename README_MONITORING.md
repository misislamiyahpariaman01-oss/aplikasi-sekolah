Monitoring features added:

- Filter logs by type and search text
- Download all logs as JSON
- Open log detail modal with copy and download per-log

Usage:
- Open the Monitoring page in the app.
- Use the filter dropdown and search box to narrow logs.
- Click "Download Logs" to save all logs as a JSON file.
- Click "Detail" on a log to open the modal; use "Copy" to copy message or "Download" to save single-log JSON.
